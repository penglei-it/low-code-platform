## 🚀 快速开始

### 1. 环境要求
- Node.js >= 18.0.0
- npm >= 9.0.0
- 现代浏览器（Chrome、Firefox、Safari、Edge）

### 2. 安装依赖
```bash
# 在项目根目录执行
npm install
```

### 3. 启动项目
```bash
# 同时启动前端和后端
npm run dev

# 或分别启动
npm run dev:frontend  # 前端：http://localhost:5173
npm run dev:backend   # 后端：http://localhost:3001
```

### 4. 访问应用
- 前端: http://localhost:5173
- 后端: http://localhost:3001

详细步骤请参考 [快速开始指南](./快速开始.md)
