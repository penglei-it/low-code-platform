## Getting Started

### Prerequisites
- Node.js 18+
- PostgreSQL 14+
- Redis 6+
- Docker (optional)

### Installation

```bash
# Install dependencies
npm install

# Start development server
npm run dev

# Start backend
cd backend && npm run dev
```
