## 三、测试验证方法

### 功能测试
1. **拖拽测试**
   - 从组件库拖拽组件到画布
   - 验证组件正确显示
   - 验证组件可被选中

2. **属性编辑测试**
   - 选中组件
   - 修改属性面板中的属性
   - 验证组件实时更新

3. **AI功能测试**
   - 输入自然语言描述

## 五、验证步骤

### 快速验证流程
1. 启动前端：`cd frontend && npm run dev`
2. 启动后端：`cd backend && npm run dev`
3. 访问 http://localhost:3000
4. 进入可视化编辑器
5. 从组件库拖拽组件到画布
6. 选中组件，在属性面板修改属性
7. 使用AI助手生成代码
8. 验证所有功能正常工作
