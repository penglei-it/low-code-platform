# 执行回归测试说明

## 前置条件

在开始测试之前，请确保：

1. **服务已启动**
   - 前端服务运行在 http://localhost:3000
   - 后端服务运行在 http://localhost:3001

2. **依赖已安装**
   - 前端依赖：`cd frontend && npm install`
   - 后端依赖：`cd backend && npm install`

## 快速启动服务

**方式1：使用启动脚本（推荐）**
```bash
# 双击运行
启动项目.bat
```

**方式2：手动启动**
```bash
# 终端1 - 后端
cd backend
npm run dev

# 终端2 - 前端
cd frontend
npm run dev
```
