### 互联网访问

1. **【Cloudflare Tunnel - 前端】**窗口

2. **【Cloudflare Tunnel - 后端】**窗口
   - 显示后端 API 的互联网访问地址
   - 格式：`https://yyy-yyy-yyy.trycloudflare.com`

## 📝 使用说明

### 访问前端应用

1. **本地访问**：在浏览器打开 `http://localhost:4173`
2. **互联网访问**：使用前端隧道窗口显示的地址

### 访问后端 API

1. **本地访问**：`http://localhost:3001/api/health`
2. **互联网访问**：使用后端隧道窗口显示的地址 + `/api/health`

### 分享给其他人

将 Cloudflare Tunnel 窗口显示的地址分享给其他人，他们即可访问您的应用！

## ⚠️ 重要提示

1. **保持窗口运行**：关闭任何服务窗口都会导致服务停止
2. **地址会变化**：每次重启 Cloudflare Tunnel，随机域名会变化
3. **等待地址生成**：Cloudflare Tunnel 地址可能需要几秒钟才能生成
